Installed Apps Page

Create a new page called Installed Apps.

Purpose

This page should contain the current apps polled from intune previously shown on the Dashboard.

Features
Move all currently polled apps from the Dashboard to this page
Display the installed or discovered app inventory in a dedicated operational view
Allow users to review the list of apps currently present in the environment
Display

Use a consistent tile/card or list-based format that aligns with the rest of the product design. Each entry should include, where available:

App logo
App name
Version
Publisher
Device count or install footprint
Details button
Behavior
This page is for inventory visibility and review
Keep this separate from recommendation logic and deployment-ready packaging
